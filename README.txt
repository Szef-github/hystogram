INSTALL
- Собирается через стандартный make. 
- Если библиотеки и заголовочные файлы лежат не там, где их ожидает Makefile, надо в Makefile прописать правильные пути, или добавить их с помощью переменных окружения
- Используется qt4 и с++11(14)
- Приложение не содержит платформозависимого кода, поэтому при аккуратной сборке будет работать на любой платформе, поддерживающей STL и QT
- Типы входных и выходных файлов выводятся из расширения, поддерживаются типы заложенные в QImage. Проверял .jpg, .bmp, .png для входа и выхода. При желании альтернативный формат может быть добавлен с помощью плагинов (QT по крайней мере обещает)
